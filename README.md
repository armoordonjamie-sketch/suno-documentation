# Suno AI Documentation

Comprehensive documentation for Suno AI music generation, including prompt guides, style references, and best practices.

## Contents

This repository contains documentation files covering:

- **Genre Lists**: Complete lists of music genres compatible with Suno
- **Style Prompts**: Style descriptors for mood, texture, rhythm, and more
- **Voice Tags**: Vocal characteristics and manipulation tags
- **Instrumental Tags**: Instrument descriptions and customization
- **Song Structure**: Tags for structuring songs ([Verse], [Chorus], etc.)
- **Artist Styles**: Style mappings for famous artists
- **Tutorials**: Step-by-step guides for creating different music styles
- **Common Problems**: Solutions to frequent issues
- **Best Practices**: Tips and techniques for better results

## Usage

These documentation files are designed to be used as reference material for:
- AI agents generating Suno prompts
- Developers building Suno integration tools
- Users learning how to create better prompts
